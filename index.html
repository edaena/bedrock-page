Bedrock

Tools and Infrastructure for Kubernetes Clusters

GitHub Project
Bedrock is a collection of tools and infrastructure definitions combined in a secure and auditable GitOps workflow that is integrated in a Kubernetes Cluster. Bedrock was designed from learnings from the cloud native community and from deploying and operating applications and Kubernetes clusters.
